# Извлечение структурированной информации из новостей

Задача: извлечь из новостных статей информацию о людях (имена, даты рождения, места рождения).

## Структура данных

```python
@dataclass
class Entry:
    name: str
    birth_date: Optional[str]
    birth_place: Optional[str]
```

## Реализация

Используется библиотека **Yargy** для извлечения имен из текста. Yargy работает на основе морфологического анализа и позволяет задавать грамматические правила.

### Грамматики Yargy

```python
Person = fact('Person', ['first', 'middle', 'last'])

# Полное имя: Имя Отчество Фамилия
NAME_FULL = rule(
    gram('Name').interpretation(Person.first),
    gram('Patr').interpretation(Person.middle),
    gram('Surn').interpretation(Person.last)
).interpretation(Person)

# Короткое: Имя Фамилия
NAME_SHORT = rule(
    gram('Name').interpretation(Person.first),
    gram('Surn').interpretation(Person.last)
).interpretation(Person)
```

Даты и места рождения извлекаются с помощью регулярных выражений.

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
python extract_yargy_full.py
```

Программа обрабатывает файл `news.txt.gz` и создает `extracted_yargy_full.txt` с результатами.

### Формат входных данных

```
категория <tab> заголовок <tab> текст
```

### Пример выходных данных

```
Имя: Владимир Высоцкий
Дата рождения: 25.01.1938
Место рождения: Москва
------------------------------------------------------------
Имя: Борис Гребенщиков
------------------------------------------------------------
```

## Примеры результатов

**С датами/местами:**
- Марианна Вертинская (30.11.1943, Москва)
- Николай Караченцов (27.10.1944)
- Эльдар Рязанов (18.11.1927)

**Только имена:**
- Владимир Путин
- Дмитрий Медведев
- Борис Гребенщиков
- Алексей Кудрин

## Статистика

При обработке 5000 статей:
- Извлечено имен: ~1200-1500
- С датами: ~20-40 (1-3%)
- С местами: ~10-20 (1-2%)

## Технологии

- **Yargy** - извлечение структурированной информации
- **pymorphy2** - морфологический анализ
- **regex** - поиск дат и мест
