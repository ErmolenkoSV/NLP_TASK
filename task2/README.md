# Классификация текстов с векторными представлениями слов

## Описание задачи

Классификация новостных текстов с использованием:
1. Модели векторного представления слов (Word2Vec, gensim)
2. Различных способов отображения документов в векторы
3. Классификатора SVM

## Реализация

### 1. Обучение Word2Vec

Модель обучается на текстах новостей:
- Предобработка: нормализация (приведение к нормальной форме), удаление стоп-слов
- Размерность векторов: 100
- Окно контекста: 5
- Алгоритм: CBOW

### 2. Отображение документов в векторы

Реализованы 4 метода:

#### Метод 1: Простое усреднение
Среднее арифметическое всех векторов слов документа.

#### Метод 2: Взвешенное усреднение по частоте
Векторы слов взвешиваются пропорционально частоте слова в документе.

#### Метод 3: Max-pooling
Для каждой размерности вектора берется максимальное значение среди всех слов документа.

#### Метод 4: TF-IDF подобное взвешивание
Векторы взвешиваются по TF-IDF метрике (term frequency × inverse document frequency).

### 3. Классификация

Используется SVM (Support Vector Machine) с линейным ядром.

```bash
pip install -r requirements.txt
```

## Использование

```bash
python text_classification.py
```

### Входные данные

Программа использует файл `../task1/news.txt.gz` с новостями.

Формат: `категория <tab> заголовок <tab> текст`

### Выходные данные

Программа выводит:
- Отчеты по классификации для каждого метода
- Сравнение точности методов
- Сохраненную модель Word2Vec (`word2vec_model.bin`)

## Особенности реализации

- **Предобработка**: приведение к нормальной форме (pymorphy2), удаление стоп-слов
- **Разделение данных**: 80% train, 20% test с сохранением пропорций классов
- **Векторное представление**: 100-мерные векторы
- **Классификатор**: SVM с линейным ядром
