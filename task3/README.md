# NER с локальными LLM и построение графа знаний

## Описание проекта

Проект реализует Named Entity Recognition (NER) с помощью предобученных небольших локальных LLM (<= 7B параметров) и сравнивает качество с альтернативными методами (CRF, spaCy, BERT, Natasha). Также включает извлечение отношений и построение графа знаний.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Установите spaCy модели (для русского):
```bash
python -m spacy download ru_core_news_sm
```

3. Установите spaCy модели (для английского):
```bash
python -m spacy download en_core_web_sm
```

## Основные возможности

- **NER с локальными LLM**: Извлечение сущностей с помощью небольших LLM (<= 7B)
- **Сравнение методов**: CRF, spaCy, BERT, Natasha
- **Извлечение отношений**: Определение связей между сущностями
- **Граф знаний**: Построение и визуализация графа знаний
- **RDF конвертация**: Преобразование в RDF формат

## Требования

- Python 3.8+
- RAM: 16GB+ (рекомендуется 32GB)
- GPU: опционально, но рекомендуется для быстрой работы
- Диск: 20GB+ для моделей и данных
